Minuto 0–5: scaffold com Next + TS + Tailwind, git init, README mínimo e `.env.example`. Minuto 5–10: tokens de cor, fontes via `next/font`, layout base com Header/Footer placeholders.

Minuto 10–15: ESLint/Prettier, alias `@/`, primeira rota real e deploy preview na Vercel. Só depois disso entro em features — nunca antes de ter pipeline de build verde.

Ter um checklist escrito (ou script) evita esquecer passos óbvios sob pressão. Repetibilidade > perfeição na primeira hora.
