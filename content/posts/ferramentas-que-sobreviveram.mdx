Testei dezenas de apps de produtividade; poucos ficaram. **Neovim** (ou Cursor quando preciso de IA), **tmux**, **ripgrep** e **lazygit** cobrem edição, terminal e Git sem context switching constante.

Para design e front, Figma continua útil para alinhar hierarquia antes de codar. **Excalidraw** para diagramas rápidos. No browser, DevTools + Lighthouse para performance e acessibilidade.

O critério de permanência é simples: a ferramenta remove atrito real ou só adiciona mais uma aba? Se não sobrevive a um mês de uso intenso, sai do setup.
