## Contexto

Redes domésticas e de pequenos escritórios crescem rápido, mas raramente têm inventário claro de dispositivos conectados. O NetAtlas nasceu para mapear hosts, serviços expostos e riscos básicos de forma visual e acessível.

## O que construí

Combinei um **dashboard Next.js** com um **agente Python** que executa varreduras via **nmap**, normaliza resultados e envia para a interface. O foco foi tornar dados técnicos legíveis para quem não vive no terminal o dia inteiro.

## Destaques técnicos

- Inventário de hosts com histórico de varreduras
- Agente local desacoplado da UI (comunicação via API)
- Insights de segurança com priorização de alertas simples
